测试智能脏状态管理是否解决了长笔记版本创建问题

测试说明

这个测试页面用于验证智能脏状态管理功能是否能正确解决长笔记版本创建问题。

问题背景：在长笔记（如三万字）中，只有修改开头部分才能创建新版本，修改中间或结尾部分时不会创建新版本。

解决方案：使用CodeMirror的智能脏状态管理，替代字符串比较。

测试步骤

  1. 点击"生成长测试内容"按钮创建模拟的长笔记
  2. 在编辑器中修改不同位置的内容
  3. 点击"模拟切换到预览"按钮
  4. 观察是否成功创建新版本
  5. 查看版本历史和测试结果
